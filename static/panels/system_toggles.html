System Toggles

Bật/tắt UI behaviour cấp tenant — không cần edit code, không cần deploy. Áp dụng ngay sau khi save (refresh page khác để thấy).

SSE Live Stream
Real-time push của scan complete events vào dashboard. Khi tắt: dashboard không auto-update khi có scan mới (user phải refresh tay). Backend poller 5s vẫn chạy server-side.
Session Timer
Counter "30:00" trên topbar + auto-logout khi hết thời gian. Reset on activity (click/key/mouse). Tắt = không có visual countdown + không auto-logout (vẫn có JWT expiry server-side).
Session Duration (phút)
Thời gian session timer (chỉ hiệu lực khi Session Timer = ON). Hợp lệ: 5–240 phút. Mặc định: 30.
Admin role required
Storage: Per-tenant trong feature_config table với feature_id='system_toggles'. JSON shape: {sse_live_enabled, session_timer_enabled, session_timer_minutes}.
Audit: Mỗi save ghi CONFIG_UPDATE:system_toggles vào audit_log.
Defaults: Mặc định all-on, 30 phút khi row chưa tồn tại.
