Configure Grafana embed

Grafana setup checklist:
  1. Set security.allow_embedding=true in grafana.ini.
  2. Configure auth proxy / shared org / anonymous access — VSP does not pass tokens.
  3. Whitelist VSP origin in security.cookie_samesite if cross-domain.
  4. Use kiosk-mode dashboards only; we append ?kiosk=tv automatically.
